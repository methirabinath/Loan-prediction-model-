# Loan Prediction using Machine Learning

## Overview

This project predicts whether a loan application should be approved using machine learning.

The project demonstrates:

- Data preprocessing
- Data cleaning
- Feature encoding
- Model training
- Model evaluation
- Loan prediction

---

## Technologies

- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
